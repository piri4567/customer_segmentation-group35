# Customer Segmentation — ML II Project

## Project Structure
```
customer_segmentation/
├── app.py                  # Streamlit interactive application
├── run_pipeline.py         # Standalone pipeline (reproduces all outputs)
├── src/
│   ├── preprocessing.py    # Feature engineering & scaling
│   ├── clustering.py       # KMeans, PCA, cluster profiling
│   ├── association.py      # Basket analysis & association rules
│   └── visualization.py   # Plot helpers (matplotlib/seaborn)
├── data/
│   ├── customer_info.csv
│   └── customer_basket.csv
├── outputs/                # Auto-generated by run_pipeline.py
│   ├── customer_clusters.csv
│   ├── cluster_summary.csv
│   ├── cluster_metrics.csv
│   └── *.png               # Visualizations
└── notebooks/
```

## Setup

```bash
pip install -r requirements.txt
```

## Run the pipeline (reproduces all outputs)

```bash
python run_pipeline.py
```

## Run the Streamlit app

```bash
streamlit run app.py
```

The app will open in your browser at `http://localhost:8501`.

## Deliverables

- `outputs/customer_clusters.csv` — every customer with their assigned cluster
- Streamlit app with 4 sections: Overview, Cluster Explorer, Campaigns, Individual Lookup
